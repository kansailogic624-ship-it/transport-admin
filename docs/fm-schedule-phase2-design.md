1. 修正画面に「前の警告 / 次の警告」— 閉じずに連続修正
2. サマリーに解消率 = （問題なし + 修正済み）/ 全警告
3. 一括処理対応構造（`bulk-action-types.ts` / `bulk-warning-actions.ts`）
4. UI確認: 1920×1080、大きい車番欄、見やすい共同作業相手、読める履歴、連続修正
